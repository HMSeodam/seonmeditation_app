## 소개

Google Gemini AI를 활용한 불교 선명상 수행 기록 프로그램입니다.  
단일 HTML 파일(`index.html`) 하나로 구성되어 있으며, 별도 설치 없이 브라우저에서 바로 실행됩니다.

\---

## 화면 구성

|화면|내용|
|-|-|
|시작|Gemini API Key 입력 후 수행 시작|
|수행 선택|오늘의 수행 유형 선택|
|五分禪瞑想|호흡 시각화 타이머와 함께하는 5분 선명상|
|省察|명상 후 하루 성찰 일기 작성|
|看話禪|공안 무작위 출제 및 AI 선지식 점검|
|收拾|수행 장소·활동·소감 기록 후 저장|
|자유게시판|수강생 간 의견·QnA·요청 게시판|

\---

## 주요 특징

* **단일 파일** — `index.html` 하나로 모든 기능 동작
* **공안 DB 내장** — 무문관·벽암록·종용록·임제록 125칙
* **AI 선지식** — Gemini API 기반 간화선 답변 점검 및 역공안 제시
* **호흡 시각화** — 들숨·날숨 애니메이션 원
* **음성 입력** — 모든 텍스트 입력 칸에서 음성 인식 지원
* **익명 게시판** — 카테고리·댓글·비밀번호 삭제 기능 포함
* **모바일 최적화** — Android·iOS 브라우저 대응

\---

## 사용 방법

1. [바로 실행하기](https://hmseodam.github.io/seonmeditation_app) 접속
2. [Google AI Studio](https://aistudio.google.com)에서 Gemini API Key 발급
3. 시작 화면에서 API Key 입력 후 **수행 시작**
